Para anunciar un paquete, sigue estos pasos:

  1. Ve a la página principal o haz clic en "Anunciar Paquete"
  2. Ingresa el nombre completo del cliente
  3. Ingresa el número de teléfono colombiano (10 dígitos)
  4. Ingresa el número de guía proporcionado por el transportador
  5. Haz clic en "Anunciar Paquete"

El sistema generará automáticamente un código de seguimiento único y enviará un SMS de confirmación al cliente.

Puedes buscar paquetes de las siguientes formas:

  • Por número de guía del transportador
  • Por código de seguimiento generado por el sistema
  • Por nombre del cliente
  • Por número de teléfono

Usa la página "Buscar" para encontrar información sobre paquetes anunciados o ya procesados.

El código de seguimiento es un identificador único de 4 caracteres generado automáticamente por el sistema cuando se anuncia un paquete.

Este código se envía automáticamente por SMS al cliente y puede usarse para:

  • Rastrear el estado del paquete
  • Consultar información del envío
  • Acceder a detalles del paquete en línea

El sistema envía automáticamente SMS de confirmación cuando:

  • Se anuncia un paquete nuevo
  • El paquete es recibido en nuestras instalaciones
  • El paquete es entregado al cliente
  • Hay cambios importantes en el estado del envío

Los SMS incluyen información relevante como el código de seguimiento, estado del paquete y enlaces para más información.
